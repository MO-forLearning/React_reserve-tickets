# 概要
 イベントチケットの予約サイトです。<br >
 React学習用のため、デザインは最小限にしています。<br >
 制作：約100時間<br >
 参考サイト：https://t.livepocket.jp/e/wyxxw

# 使用技術
- react 18.3.1
- axios 1.7.3
- node 18.17.0
- npm 9.6.7
- cors 2.8.5
- express 4.18.2
- mysql2 3.3.1
- css3

# 機能一覧
- ユーザー登録、ログイン機能
- チケット購入機能
  - ログイン時のみ購入可能
  - 残数表示
- マイページ表示機能
  - 購入済チケット一覧表示
